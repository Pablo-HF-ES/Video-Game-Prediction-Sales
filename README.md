Análisis de la Disminución del Precio de Salida de Videojuegos
Este proyecto implementa una arquitectura de Machine Learning para predecir la degradación del precio de lanzamiento de videojuegos utilizando datos históricos de ventas y promociones.

Arquitectura del Proyecto
El proyecto está estructurado siguiendo las mejores prácticas de Cookiecutter Data Science, asegurando la reproducibilidad y modularidad:

data/: Contiene los datos en sus diferentes etapas (raw, interim, processed).
models/: Almacena los modelos entrenados (.joblib) y artefactos de preprocesamiento.
src/: Código fuente modularizado para ingesta, ingeniería de características, entrenamiento e inferencia.
reports/: Informes y métricas de desempeño.
Conjunto de Datos
El conjunto de datos fue obtenido de la página web de Kaggle. El enlace al dataset se proporciona a continuación: https://www.kaggle.com/datasets/amith1707/video-game-market-price-and-revenue-dataset
